# 🔐 React Authentication System

A beautifully designed, fully responsive authentication system built with React.js featuring Login, Signup, and Forgot Password functionality.

## ✨ Features

- 🔑 **Login** - Email and password authentication
- 📝 **Signup** - Create new account with name, email, password
- 🔒 **Forgot Password** - Password reset via email
- 🎨 **Modern UI** - Glassmorphism design with Yellow/Black theme
- 📱 **Responsive** - Works perfectly on mobile, tablet, and desktop
- ⚡ **Smooth Animations** - Hover effects, transitions, and floating cards

## 🛠️ Tech Stack

- **React.js** - Frontend library
- **React Router DOM** - Page navigation
- **Bootstrap 5** - Responsive grid & components
- **Vanilla CSS** - Custom animations & glassmorphism
